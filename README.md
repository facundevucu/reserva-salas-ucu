 Sistema de Reserva de Salas - UCU

Proyecto académico desarrollado en Python (Flask) y con la ayuda de Next.js, que permite gestionar reservas de salas, participantes y sanciones dentro de la universidad.

⸻

 Tecnologías
	•	Backend: Flask + MySQL
	•	Frontend: Next.js (v0) + TailwindCSS (Parte opcional desarrollada con IA)
	•	Arquitectura modular: validaciones, ABM y reportes por entidad

⸻

 Funcionalidades
	•	ABM de participantes, salas, reservas y sanciones
	•	Validaciones de datos y control de fechas
	•	Dashboard con estadísticas generales
	•	Interfaz web simple y responsive

⸻

 Ejecución rápida

# Iniciar backend
python app.py

# Generar frontend
cd frontend
npm install --legacy-peer-deps
npm run build

Luego abrir: http://127.0.0.1:5000/ o http://localhost:5000/
